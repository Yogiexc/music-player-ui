# ⌨️ Keyboard Shortcuts Guide

## Available Shortcuts

| Key | Action | Description |
|-----|--------|-------------|
| `Space` | Play / Pause | Toggle playback state |
| `→` | Next Track | Skip to next song |
| `←` | Previous Track | Go to previous (or restart current) |
| `L` | Toggle Theme | Switch between dark/light mode* |

\* *Coming soon*

---

## Usage Examples

### Play/Pause
```
Press Space → Music starts playing
Press Space again → Music pauses
```

### Navigation
```
Press → → Skip to "Neon Lights"
Press ← → Go back to "Midnight Dreams"
Press ← (within 3 seconds) → Previous track
Press ← (after 3 seconds) → Restart current track
```

---

## Accessibility Notes

- All shortcuts work when focus is **not** on input elements
- Shortcuts are announced to screen readers
- Visual focus indicators show active element
- Keyboard navigation follows logical tab order
